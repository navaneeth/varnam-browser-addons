How to use Varnam input tool

To activate varnam input tool

  • Right click on the input area.
  • Select the desired language
  • Start typing and you can see varnam shows suggestions for the input. Pressing SPACE or ENTER will insert the selected suggestion to the input area.

Getting help

If you have any questions, please post it to varnamproject@googlegroups.com. You can also use #varnamproject on freenode for discussions

Reporting bugs

Bugs can be reported at https://github.com/navaneeth/varnam-browser-addons/issues
